## RESPONSIVE BREAKPOINTS

### Mobile-First Approach

```
Mobile (< 640px):
  - Font size: 16-18px (large for readability)
  - Padding: 16px sides
  - Stack: 1 column
  - Touch targets: 44px minimum
  - Card gap: 12px

Tablet (640px - 1024px):
  - Font size: 16-18px
  - Padding: 24px sides
  - Stack: 2 columns max
  - Gap: 16px
  - Hero buttons: stack vertical

Desktop (> 1024px):
  - Font size: 18px
  - Padding: 32px sides (container centered)
  - Grid: 2-4 columns as needed
  - Gap: 24px
  - Hero buttons: horizontal
```

## ANIMATION RECOMMENDATIONS

### Section Reveal (Scroll-triggered)
```
Initial:     opacity: 0, translateY(20px)
On View:     opacity: 1, translateY(0)
Duration:    600ms
Delay:       Based on element index
Easing:      ease-out
Already in:  Framer Motion, whileInView
```

### Card Stagger (Multiple items)
```
Parent:      container
Children:    staggerChildren: 0.1s
Timing:      sequential, 100ms apart
Duration:    500-600ms total
Easing:      "spring" for bounce feel
```

### Button Click Feedback
```
Initial:     scale: 1
Click:       scale: 0.95, duration: 100ms
Release:     scale: 1, duration: 200ms
Type:        Tap animation, physical feedback
```

### Micro-interactions
```
Loading:     Spinner animation (2s rotation)
Success:     Checkmark + fade-in
Error:       Shake animation + red flash
Notification: Slide-in from top, fade-out after 3s
```
